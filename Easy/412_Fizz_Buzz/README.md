# 412. Fizz Buzz

## Difficulty
Easy

## Problem Summary
Given an integer `n`, return a list of strings from `1` to `n` where:

- Use `"FizzBuzz"` if a number is divisible by both 3 and 5.
- Use `"Fizz"` if a number is divisible by 3.
- Use `"Buzz"` if a number is divisible by 5.
- Otherwise, use the number itself as a string.

## Concepts Used
- Loops (`for`)
- Conditional statements (`if`, `elif`, `else`)
- Modulus operator (`%`)
- Lists

## What I Learned
How to check divisibility using the modulus operator (%)
Why the condition for both 3 and 5 must be checked first
How to build and return a list in Python
How to convert numbers to strings using str()
