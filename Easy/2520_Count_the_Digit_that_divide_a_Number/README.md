# 2520. Count the Digits That Divide a Number

## Problem Statement
Given an integer `num`, return the number of digits in `num` that divide `num` evenly.

A digit divides `num` evenly if `num % digit == 0`.

## Example
  Input: num = 121
  Output: 2

Explanation:
- 1 divides 121
- 2 does not divide 121
- 1 divides 121

So the answer is 2.

Concepts Used
  Modulo Operator (%)
  Integer Division (//)
  While Loop
  Counting
